# Music-Store-DBMS-MySQL
A simple MySQL-based database system developed for a music retail business to manage branches, employees, music instruments, inventory, and sales transactions. The project demonstrates relational database design and SQL programming. 

Files in this repository include: 
1. Data Definition Language (DDL)
Database schema creation, table relationships, constraints, and database structure design.
2. Data Manipulation Language (DML)
Sample data population insertion, update, deletion, and alteration operations.
3. Data Query Language (DQL)
Business-oriented SQL queries for sales reporting, inventory monitoring, employee management, and transaction analysis.
4. Advanced Queries
Implementation of intermediate-to-advanced SQL concepts including stored procedures, subqueries, joins, conditional logic, and parameterized search functions.
