# Music-Store-DBMS-MySQL
Music Store Database Management System built with MySQL featuring database design, data manipulation, data querying, and advanced SQL operations for inventory and transaction management.

1. Data Definition Language (DDL)
- Database creation
- Table creation
- Primary Key and Foreign Key constraints
- CHECK constraints and data validation
- ALTER TABLE operations
- Relational database design

2. Data Manipulation Language (DML)
- Data insertion using INSERT statements
- Data modification using UPDATE statements
- Data deletion using DELETE statements
- Population of sample transactional data

3. Data Query Language (DQL)
-Filtering and sorting data
- String manipulation functions
- Date formatting functions
- Aggregate functions (COUNT, SUM, AVG, MIN, MAX)
- Grouping and ordering results
- Multi-table JOIN operations

4. Advanced SQL Concepts
- Stored Procedures
- Subqueries in WHERE, SELECT, and FROM clauses
- CASE WHEN expressions
- LEFT JOIN and RIGHT JOIN
- COALESCE function
- Parameterized search procedures based on business questions
